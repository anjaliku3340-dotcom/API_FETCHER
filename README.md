# 📌 API User Fetcher (Python CLI Project)

## 🚀 Overview
API User Fetcher is a simple Python command-line application that retrieves user data from a public API and displays it in a clean, readable format. It also allows users to search for specific users by name.

This project demonstrates API handling, error management, and basic data processing using Python.

---

## ✨ Features
- 🔗 Fetch user data from a public API
- 📋 Display users in a formatted list
- 🔍 Search users by name (case-insensitive)
- ⚠️ Robust error handling for network and JSON issues
- 🧩 Clean and modular code structure

---

## 🛠️ Technologies Used
- Python
- requests library
